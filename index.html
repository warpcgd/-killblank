快速开始🦴

# 1. 安装

  npm i @kiilblank/prerender -D

# 2. 配置

// webpack.config.js
const prerender = require('@killblanks/prerender')

export default {
  ...
  plugins: [new prerender()]
  ...
}

# 3. 使用@killblanks/skeleton-ext

# 4. 启用PRERENDER_SKELETON

 1. 在Chrome console中输入`PRERENDER_SKELETON`启动骨架屏预览
